Кексобукинг

Выпускной проект Профессионального JS, уровень 1.

Кексобукинг – сервис размещения объявлений о сдаче в аренду недвижимости в центре. Пользователям предоставляется возможность размещать объявления о своей недвижимости или просматривать уже размещённые объявления.

Данные о размещенных объявлениях загружаются с сервера, на их основе отрисовываются маркеры на карте. При клике на маркер отображается карточка, содержащая подробную информацию об объявлении. Данные можно отфильтровать по типу жилья, его стоимости, количеству комнат и гостей, а также по наличию дополнительных удобств.

При размещении своего объявления заполняемая форма проходит валидацию.

Значение поля «адрес» автоматически выставляется при перемещении главной метки по карте.

После заполнения всех данных, при нажатии на кнопку «Опубликовать», все данные из формы с помощью AJAX-запроса отправляются на сервер.

При успешной отправке формы, страница, не перезагружаясь, переходит в изначальное неактивное состояние.

Если при отправке данных произошла ошибка запроса, показывается соответствующее сообщение.

Code and magick

Учебный проект Профессионального JS, уровень 1.

В проекте реализовано:

  • при нажатии на аватарку открывается окно настроек персонажа;
  • у волшебника можно поменять цвет мантии, глаз и фаербола;
  • с сервера загружаются данные о других персонажах; при изменении настроек персонажа данные о волшебниках фильтруются и отрисовываются в окне похожих персонажей;
  • окно настроек можно перемещать по экрану;
  • по окончании игры выводится окно со статистикой.

Тестовое задание Avito

Реализация тестового задания Авито.

Одностраничный проект просмотра списка объявлений по продаже автомобилей, ноутбуков, недвижимости и фотоаппаратов от частных лиц и компаний и показом подробной информации по каждому из них.

Данные загружаются с сервера. На основании загруженных данных формируется список товаров. С помощью обратного геокодирования API Яндекс.Карт отображается адрес продажи.

Список товаров можно отфильтровать по категории и стоимости, а так же настроить сортировку – по популярности, по уменьшению или увеличению цены.

При выборе категории по каждой из них появляются дополнительные контролы для фильтрации.

Товары можно добавлять в избранное и просматривать список избранных товаров отдельно. Фото товаров меняются раз в несколько секунд.

По каждому объявлению можно посмотреть подробную карточку товара, с картой, рейтингом и названием продавца.

Более подробно реализованные требования можно посмотреть в техническом задании.
